# 3DCP_research
Interactive literature graph and table for 3D concrete printing (3DCP).

## GitHub Pages
This repo is GitHub Pages ready. Point Pages at the `main` branch (root) and the app will be served from `index.html`.

## Usage
- Open `index.html` locally or via GitHub Pages to explore the interactive graph/table.
- Data source used to build the page is included as `3DCP_Full_compositions.xlsx`.
